

As a user, I can view all the inventories as well as search certain inventories by names, so that I can manage them.

Use Case Driven Development

Main Scenario

- Pre Condition
1. The user is on the inventories page.

- Post Condition
2. The user sees the inventories that meet the search condition.

- Detailed Steps
1. The user enters a few letters in the search box.
2. The user clicks on the view button.

---------------------------------------

Alternative Scenario A
- Post Conditions
2a. The user sees all the inventories

- Detailed Steps
1a. The user leaves the search box empty.
